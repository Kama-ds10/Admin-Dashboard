# Admin-Dashboard

Project Overview

The SHIFT Code Admin Dashboard serves as a foundational project for learning front-end web layout and responsive design.
It features a structured dashboard layout with:

A Sidebar for navigation

A Topbar (Header) with a greeting and action buttons

A Main content area with project cards

A Right column for announcements and trending topics

This project demonstrates CSS Grid mastery, responsive behavior, and consistent design principles.
